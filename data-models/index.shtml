Data models exercises

Cars 1

Create a Car class, with whatever attributes you like, but make one of the attributes weight. Give the class a render() function. Make a page that shows data about five cars. Base it on the first dog class.

Cars 2

Use an array for your cars, as in the second dog example. Write code that shows all of the car data.

Show the total weight of all the cars. Write code that works no matter how many cars are in the array.

Cars 3

Add an id attribute to Cars.

Make a collection class, with addNew(), render(), and delete functions. Use the third dog example as a reference.

Cars 4

Add an interactive display of deets, like the fourth dog example.

Change it so that the row is highlighted and the deets are shown when the user moves the mouse cursor over a car, without having to click. If s/he moves the cursor off the table, the deets go away, and nothing is highlighted.

Cars 5

Make a car editing app, like the fifth dog example.

In the line that shows the name of the car collection, show the number of cars in parens after the collection name. Like Dream Fleet (8 cars).
